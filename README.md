# Cider
Checks if domains/ip addresses are within CIDR addresses   

Useful For   
- Checking if targets are within scope
- Checking if a domain/url is within the mass scan DONOTSCAN List
- Preventing Trouble with known government/military addresses

Ive Included a Do Not Scan list that i compiled from various sources accross the web, these are ip ranges that should be left alone.
